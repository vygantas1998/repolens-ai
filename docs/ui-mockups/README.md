# RepoLens UI Mockups

These are standalone HTML mockups only. They do not change the current React app.

## Options

1. `01-command-center.html` - dark, premium developer power-tool style with a cyber command-center feel.
2. `02-canvas-inspector.html` - bright, visual canvas style with a right-side inspector, closer to a design/devtool product.
3. `03-dark-canvas-inspector.html` - dark version of option 2 with the same canvas + inspector structure.
4. `04-architecture-node-options.html` - focused alternatives for the architecture canvas node/card design.
5. `05-architecture-canvas-concepts.html` - larger canvas improvement concepts with linked file tree, lanes, and pseudo-3D map.
6. `06-city-map-concept.html` - experimental future concept showing the codebase as an isometric city map.
7. `07-city-map-v2.html` - stronger city-map concept with clearer building faces, streets, districts, landmarks, and inspector rules.
8. `08-interactive-3d-city.html` - interactive panning/zooming city mockup with folder area and building height semantics.
9. `09-rotating-3d-city.html` - rotating CSS 3D city mockup with actual cuboid building faces and stronger object depth.
10. `10-r3f-code-city.html` - real Three.js / React Three Fiber prototype with orbit, zoom, pan, shadows, lights, and clickable 3D buildings.
11. `11-ux-density-improvements.html` - UX refinements for reducing panel density, adding mode tabs, and clarifying curated important files versus all node files.

Open any file in a browser and choose the direction you prefer. After you choose, the selected style can be implemented in the actual React UI.
